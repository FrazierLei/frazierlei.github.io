cout 格式化输出

August 6, 2021

在整值型高位补0使其位数为m

cout << setfill('0');
cout << setw(4) << 666 << "\n"<< setw(5) << 666;

/*
输出：
0666
00666
*/

保留m位有效数字/小数

double i = 3.1415926;
cout << setprecision(2) << i << endl; 
cout << setprecision(2) << fixed << i;

/*
输出
3.1
3.14
*/
