Leetcode 字典序排数

March 2, 2022

原题

题干要求时间复杂度为 $\mathrm{O}(n)$，提前否定了最容易想到的蛮力法——先初始化再排序。

不过还是试验一下蛮力法的运行时间。

蛮力法

代码：

class Solution {
public:
    vector<int> lexicalOrder(int n) {
        vector<int> v(n);
        iota(v.begin(), v.end(), 1);
        sort(v.begin(), v.end(), [](const int a, const int b) {
            return to_string(a) < to_string(b);
        });
        return v;
    }
};
