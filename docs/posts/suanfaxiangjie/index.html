推荐一下 日沉云起 大佬的这本《算法详解》

October 8, 2021

很早就在98上看到校友 日沉云起 （好像和我还是同届的研究生，人和人之间的差距可太大了）出版了一本《算法详解（C++11语言描述）》的消息。

7月底的时候买了这本书，然后用了一个多月时间把这本书看了两遍，做了大部分的例题和习题，很有收获。

第一遍看的时候看得比较匆忙，跳过了第8章、第15章和第16章。每章只做了例题和每种题型下的一道习题。

第二遍看的时候把每章后面的习题都做完了，又把第15章「复杂模拟」中的题目做了几道。

国庆期间我看完了《21天学通C++（第8版）》（这本书也是一本很好的 C++ 入门书，比《C++ Primer》对新手更加友好）和《C++语言导学》（C++创作者的书，很薄，只有200页）

昨天晚上回到杭州，用一个晚上的时间把之前跳过的第8章和第16章看完了，又把「泛型算法」中自定义排序这一部分重新看了一遍，这部分内容如果没有学过面向对象的知识很难掌握。

下面我列举一下这本《算法详解》好在哪里：

  1. 不需要 C/C++ 的基础，适合初学者和转码选手。
  2. 全书采用 C++11 的语法，避开了传统算法书籍采用的 C+STL 的模式。而且补充了部分 C++14/17 的语法知识点。
  3. 内容充实，编排得当。「顺序容器」、「泛型算法」、「关联容器」这几章的内容是解题的利器，而且讲完语法马上用例题展示其用法，既能加深理解，又不会觉得枯燥。
  4. 代码质量很高。本书对应的 GitHub repo 中有 PAT 甲级和乙级全部习题的解答，部分题目还有多种解答，而且比网上搜索到的大部分解答都要简洁明了。
  5. 习题经过详细分类。可以在 解题目录 中查找每道题考查的知识点，方便分类刷题。

学到「图论」这一部分的时候我遇到了一个现在看很显然的问题，在 Issues 中提出后很快得到了回复。

后来我在 98 的私信中询问作者 C++ 学习路线的问题，也很快收到了作者的回复和鼓励。

我大二时候选修过计算机学院的「数据结构基础」课程，当时用的是 C 语言，但是我当时学的很不扎实，虽然期末考试考的还不错，但是链表后面的内容我几乎都忘光了。

大三时候选修了信电学院的「算法分析」课程，整门课程都是在纸上谈兵（信电很多课都是这个样子，至少我本科时候如此），整个学期我几乎没写一行代码，效果可想而知。

这本书给了我重新拾起算法和 C++ 的机会，感谢作者和这两个月专心学习的自己。
