测试 MathJax 的渲染效果

July 23, 2021

Hugo 本身是不支持 Latex 的渲染的，所以我早期的一些包含较多公式的文章无法显示。

根据网上几个教程，用 MathJax 实现了这个功能。

之前含有公式的文章都是在 Wordpress 上用 Katex 写的，迁移过来还需要一些额外的工作，之后有空再完成好了。

主要参考了：

  • 在Hugo中使用MathJax

  • Render LaTeX math expressions in Hugo with MathJax 3

  • 通用 LaTeX 数学公式语法手册
